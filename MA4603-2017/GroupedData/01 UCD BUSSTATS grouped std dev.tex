\begin{frame}
\frametitle{Grouped Data - Standard Deviation}
{
\LARGE
The mean of the grouped data is computed as follows:

\[ \bar{x} = \frac{\sum f_ix_i}{\sum f} \]
\bigskip

$f_i$ is the frequency for interval $i$,\\
$x_i$ is the midpoint for interval $i$.

}
